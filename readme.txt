== Description ==

This is a shortcode in post content which enables you to share your favorite posts with url only. Please note that this plugin use outside free service "http://capture.heartrails.com/" capturing the screenshot of the website. Please also note that hatebu outside service (free) shows visitors how many bookmarked. These are optional and you can turn off as follows:

[sharepost url="(url of your favorite website)" img="(true|false)" hatebu="(true|false)"]

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. 'Share Post Shortcode' menu in Settings
